# 🔍 Python BLE Scanner

A lightweight, efficient Python-based Command Line Interface (CLI) utility designed to scan, discover, and analyze nearby Bluetooth Low Energy (BLE) devices. Built on top of modern asynchronous Python, this tool provides real-time tracking of BLE advertisements and device metadata.

---

## ✨ Features

*   **Asynchronous Scanning:** Uses `asyncio` for non-blocking, high-performance device discovery.
*   **Real-Time Metadata:** Captures critical device details including:
    *   MAC / UUID Addresses
    *   Device Name (Local Name)
    *   RSSI (Signal Strength indication)
    *   Advertisement Data & Service UUIDs
*   **Clean CLI Output:** Formatted terminal interface for easy readability during live scans.
*   **Cross-Platform:** Works seamlessly across Windows, macOS, and Linux.
